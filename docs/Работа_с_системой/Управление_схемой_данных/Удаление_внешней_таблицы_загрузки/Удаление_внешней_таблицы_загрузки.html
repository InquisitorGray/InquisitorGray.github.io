Удаление внешней таблицы загрузки

Чтобы удалить внешнюю таблицу загрузки, выполните запрос DROP UPLOAD EXTERNAL TABLE (см. пример ниже). При успешном выполнении запроса внешняя таблица удаляется из логической схемы данных.

Пример

-- выбор базы данных sales по умолчанию
USE sales

-- удаление внешней таблицы загрузки
DROP UPLOAD EXTERNAL TABLE sales_ext_upload
