Удаление внешней таблицы выгрузки

Чтобы удалить внешнюю таблицу выгрузки, выполните запрос DROP DOWNLOAD EXTERNAL TABLE (см. пример ниже). При успешном выполнении запроса внешняя таблица выгрузки удаляется из логической схемы данных.

Пример

-- выбор базы данных sales по умолчанию
USE sales

-- удаление внешней таблицы выгрузки
DROP DOWNLOAD EXTERNAL TABLE sales_ext_download
