Загружаемые типы данных

Система поддерживает загрузку типов данных Avro, перечисленных в таблице ниже. Для каждого типа данных Avro указан соответствующий логический тип данных.

Подробнее о типах данных Avro см. в официальной документации Apache Avro.

Загружаемый тип данных Avro Логический тип данных
boolean BOOLEAN
string VARCHAR (n)
long BIGINT
double DOUBLE
float FLOAT
(int) date DATE
(long) time-micros TIME (precision)
(long) timestamp-micros TIMESTAMP (precision)
