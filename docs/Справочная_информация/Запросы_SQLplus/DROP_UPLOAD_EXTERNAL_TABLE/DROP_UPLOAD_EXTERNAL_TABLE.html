DROP UPLOAD EXTERNAL TABLE

Запрос позволяет удалить внешнюю таблицу загрузки из логической базы данных.

В ответе возвращается:

  • пустой объект ResultSet при успешном выполнении запроса;
  • исключение при неуспешном выполнении запроса.

Синтаксис

DROP UPLOAD EXTERNAL TABLE [db_name.]ext_table_name

Параметры

  • db_name — имя логической базы данных, из которой удаляется внешняя таблица загрузки. Опционально, если выбрана логическая БД, используемая по умолчанию;
  • ext_table_name — имя удаляемой внешней таблицы загрузки.

Пример

DROP UPLOAD EXTERNAL TABLE sales.sales_ext_upload
