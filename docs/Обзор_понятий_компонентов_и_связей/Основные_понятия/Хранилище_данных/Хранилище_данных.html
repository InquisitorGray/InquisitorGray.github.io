Хранилище данных

Физическое хранилище данных (далее — хранилище) — совокупность СУБД различного типа, в которых хранятся данные логических таблиц. Данные в хранилище хранятся в соответствии с физической схемой данных.

Система предоставляет единый интерфейс взаимодействия с хранилищем в виде логической базы данных.
